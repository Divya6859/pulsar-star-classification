# 🌌 Pulsar Star Classification

## 📌 Overview
This project classifies pulsar stars based on astronomical signal data using machine learning.

## 🚀 Features
- Data preprocessing and feature analysis  
- Classification using Random Forest, SVM, and XGBoost  
- Model comparison  

## 📊 Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  

## 📈 Evaluation Metrics
- F1-score  
- Recall  
- Confusion Matrix  
- ROC Curve  

## 📂 Dataset
- Pulsar star dataset  

## ▶️ How to Run
1. Load dataset  
2. Train models  
3. Evaluate performance  

## 📌 Result
The model identifies pulsar stars with strong classification performance.
